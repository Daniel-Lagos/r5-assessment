# r5-assessment

## Getting Started

---

First, if you use MacOS you could be install pnpm with brew

```bash
brew install pnpm
```

In other hand, you could check the guide [pnpm](https://docs.volta.sh/guide/getting-started).

<br>
Second, run the development server:

```bash
pnpm dev
```
